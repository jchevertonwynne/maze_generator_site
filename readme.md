# Maze generator site

Simple little maze maker + display site

## Install required modules

`$ pip install -r requiremnts.txt`\

## Make `secret.py`

`secret.py:`

`secret = "YOUR_STRING_HERE>"`

## Setup database

`$ cd application`

`$ chmod +x database.py`

`$ ./database.py`

## Run application

`$ chmod +x application.py`

`$ ./application.py`
